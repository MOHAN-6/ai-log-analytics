# 📊 AI-Ready Log Analytics & Monitoring System

AI-powered log analytics with real-time anomaly detection and severity classification.

## ✨ Features

- 🧠 **AI Severity Classification** (INFO, WARNING, ERROR, CRITICAL)
- 🚨 **Anomaly Detection** with Isolation Forest
- 📊 **Real-Time Dashboard**
- 📥 **Log Ingest & Analysis**

## 🛠️ Tech Stack

- **Backend:** Flask + scikit-learn
- **Frontend:** HTML, CSS, JavaScript
- **AI:** Isolation Forest, Naive Bayes, TF-IDF

## 🚀 Quick Start

```bash
pip install -r requirements.txt
python app.py
